# 灵感快收公开版产品规则

版本日期：2026-07-17

## 固定交互

```text
长按 iPhone Action 键
-> 立即开始中文语音听写
-> 用户手动点完成
-> 弹出唯一一个可编辑文本框
-> 文本框默认带入语音识别结果
-> 用户可键盘修正并点完成
-> 后台静默处理
```

听写停止方式必须为 `On Tap / 在轻点时`。

## 明确不要

- 不要在语音前弹输入方式选择。
- 不要出现第二个文本框。
- 不要出现成功码、重试结果或技术通知。
- 不要使用 iCloud 文件删除、覆盖或清空动作。
- 不要因为用户说话停顿而自动结束听写。
- 不要依赖 Mac 开机、同一 Wi-Fi 或本地后台。

## 数据流

所有内容统一写入飞书多维表格。

- 普通灵感、素材、备忘：只进入飞书。
- 可完成、可打勾的事项：进入苹果提醒事项，并进入飞书。
- 需要占用时间的安排：进入苹果日历，并进入飞书。

第一版采用明确关键词进行本机分流，不调用复杂 AI 接口。

## 默认字段原则

采集阶段避免过度判断：

```text
下一步用途 = 待判断
实现路径 = 待判断
```

来源链接只能填写真实的 `http/https` 链接，不能把普通口述文字当作链接。
