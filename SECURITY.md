# 安全说明

这个项目不会提交任何真实的飞书凭证。

## 不要公开的内容

- 飞书 App Secret
- 个人使用的 `app_token` 和 `table_id`
- 已生成的 `.shortcut` 文件
- 真实飞书记录和导出数据
- 包含个人日历、提醒事项或业务数据的截图

## 为什么不发布生成后的快捷指令

本项目采用 iPhone 直接调用飞书 OpenAPI 的方式。生成器会把 App ID、App Secret、
`app_token` 和 `table_id` 写进快捷指令，以便手机离开 Mac 后仍能独立运行。

因此，生成后的 `.shortcut` 属于私人配置文件，不能上传公开仓库或直接分享给他人。
公开的是生成器、空白配置示例和安装文档。

## 泄露后的处理

如果 App Secret 或生成后的快捷指令曾被公开：

1. 立即在飞书开放平台重置 App Secret。
2. 删除公开文件和历史附件。
3. 重新把新 Secret 存入 macOS 钥匙串。
4. 重新生成并导入私人快捷指令。

发现安全问题时，请不要在公开 Issue 中粘贴任何凭证。
